`floyd_warshall.cpp` 使用说明

1. 功能
- 计算图中任意两点之间的最短路径。
- 适合需要多次查询最短路的场景。
2. 输入输出方式
- 文件输入：`floyd_warshall.in`
- 文件输出：`floyd_warshall.out`
3. 输入格式
- 先输入点数 $n$ 和边数 $m$。
- 然后输入 $m$ 条边，每条边包含三个正整数 $u$，$v$，$w$，使用空格分开。
	- $u$：起点
	- $v$：终点
	- $w$：边权
4. 输出格式
- 第 $i$ 行第 $j$ 列表示从节点 $i$ 到节点 $j$ 的最短路径距离。

`dijkstra.cpp` 使用说明

1. 功能
- 计算图中一个起点到其他起点的最短路径。
- 适合需要从一个点出发多次查询不同终点最短路的场景。
2. 输入输出方式
- 文件输入：`dijkstra.in`
- 文件输出：`dijkstra.out`
3. 输入格式
- 先输入点数 $n$ ，边数 $m$ 和起点 $s$。
- 然后输入 $m$ 条边，每条边包含三个正整数 $u$，$v$，$w$，使用空格分隔。
	- $u$：起点
	- $v$：终点
	- $w$：边权
4. 输出格式
- 每一行表示从起点到该点的最短路径长度。
